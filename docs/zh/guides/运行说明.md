# CART决策树测试运行说明

## 安装 Rust

如果您的系统还没有安装 Rust，请按照以下步骤安装：

### Windows 安装方法：

1. 访问 https://www.rust-lang.org/tools/install
2. 下载并运行 `rustup-init.exe`
3. 按照安装向导完成安装
4. 重启终端或运行 `refreshenv` 刷新环境变量

或者使用 PowerShell 一键安装：
```powershell
Invoke-WebRequest https://win.rustup.rs/x86_64 -OutFile rustup-init.exe
.\rustup-init.exe
```

## 运行测试

安装 Rust 后，在项目目录下运行：

```bash
cargo run --bin cart_test
```

或者先编译再运行：

```bash
cargo build --bin cart_test
cargo run --bin cart_test
```

## 测试内容

程序包含三个测试：
1. **分类树测试** - 测试二分类问题
2. **回归树测试** - 测试回归问题
3. **多特征分类测试** - 测试多特征分类问题

每个测试都会：
- 显示训练数据
- 训练决策树
- 显示训练后的树结构
- 进行预测并显示结果
